What is AJAX?

AJAX stands for Asynchronous JavaScript and XML. It is a technique used to create dynamic and interactive web applications by allowing data to be sent and received asynchronously without interfering with the display and behavior of the existing page.

Why do you put your JS Code in a separate file?

Putting JavaScript code in a separate file helps to keep the HTML file clean and organized. It also allows for better reusability and maintainability of the code, as the same JavaScript file can be included in multiple HTML files.

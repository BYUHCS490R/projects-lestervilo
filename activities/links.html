  2. What is the root? What is the relationship between folder and file names and the path?
    • The root is the top-level directory of a file system.
    • The relationship between folder and file names and the path is that the path specifies the location of a file or folder within the directory structure, starting from the root and including all parent folders leading to the target file or folder.
  3. What is a default document?
    • A default document is the file that is served when a directory is requested.

  6. List the 3 rules with relative links.
    1. Down a Folder - use the Folder Name and Filename.
    2. Up a Folder - use ../ and then the Folder Name and Filename.
    3. Start Over - use / and then the Folder Name and Filename.

Internet

The Internet is a global network that connects all the computers all over the world.

Web

The web is a tab in any browser that has links that a user can view.

How are the Internet and Web different?

The internet is a hardware that connects the computers, while the web is a software that runs on the network.

ASCII and Unicode

ASCII (American Standard Code for Information Interchange) is like an alphabet however this mainly help turn the numbers to a letter.

What is UTF-8?

UTF-8 (Unicode Transformation Format - 8-bit) helps to store and display text all over the internet.

URL

A URL (Uniform Resource Locator) is the address used to access resources on the Internet.

Domain Name

A domain name is a human-readable address like google.com.

HTTP

HTTP (Hypertext Transfer Protocol) allows the web browser to send data back and forth to the web server.

HTTPS

HTTPS (Hypertext Transfer Protocol Secure) is a secure version of HTTP because it encrypts the data being sent between the web browser and the web server.

IP Address

An IP address (Internet Protocol address) is a unique numbers assigned to evry device to identify it on the internet.

DNS

DNS (Domain Name System) is a system that translates domain names into IP addresses.

W3C Recommendations

W3C (World Wide Web Consortium) ensures that web standards are followed to promote long-term growth of the web.

XML, XHTML, HTML, HTML5

XML (eXtensible Markup Language), XHTML (eXtensible Hypertext Markup Language), HTML (Hypertext Markup Language), and HTML5 are markup languages used to create web pages and define the structure of web content.

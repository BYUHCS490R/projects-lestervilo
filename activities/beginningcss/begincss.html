Questions:

Explain the difference between inline, embedded, and external CSS.

Inline css is adding the style within the line, embedded css is within the style tag in the header, and the external css is in a separate file and linked to the HTML document.

Explain why you should only use external CSS.

Because it helps keep the html clean and easier to maintain.

What is a selector used for?

A selector is used to select the HTML elements you want to style.

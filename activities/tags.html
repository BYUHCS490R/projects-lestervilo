Common HTML Tags

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Anchor (link)
  • <img> - Image
  • <ul> - Unordered List
  • <ol> - Ordered List
  • <li> - List Item
  • <div> - Division/Section
  • <span> - Inline Container
  • <br> - Line Break

Basic Structure Tags

  1. <html> - Root element
  2. <head> - Metadata container
  3. <title> - Document title
  4. <body> - Document body

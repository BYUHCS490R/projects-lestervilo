Answer the following questions:

  1. What is the difference between block tags and inline tags?
    2. Block tags take up the full width of their container, while inline tags only take up as much width as necessary.
  2. What is the margin, border, and padding?
    3. Margin is the space outside the border, border is the line around the element, and padding is the space inside the border.
  3. What are units of measurement you can use in CSS?
    4. Pixels (px), ems (em), rems (rem), percentages (%), and others like ex, ch, vw, vh.
  4. When would you use a figure tag?
    5. The figure tag is used to group content like images, diagrams, or code examples with a caption.

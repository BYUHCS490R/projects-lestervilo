Questions

  1. What is normal flow in HTML?

    2. Normal flow is the default way that HTML elements are laid out on a page. Block-level elements stack vertically, while inline elements flow horizontally within their containing block.

  2. What does float do? Why do you need to clear the float?

    3. Float allows elements to be taken out of the normal flow and positioned to the left or right of their container. Clearing the float is necessary to prevent subsequent elements from wrapping around the floated element, which can cause layout issues.

  3. What do absolute, fixed, and relative do?

    4. Absolute positioning removes an element from the normal flow and positions it relative to its nearest positioned ancestor. Fixed positioning removes the element from the normal flow and positions it relative to the viewport, so it stays in place when scrolling. Relative positioning allows you to adjust an element's position relative to its normal position in the flow.

  4. How can you make nice navigation buttons for mobile devices?

    5. To create nice navigation buttons for mobile devices, you can use CSS media queries to adjust the layout and styling based on the screen size. Consider using larger touch targets, stacking navigation items vertically, and using a hamburger menu for a cleaner look.

  5. Where can you find videos to review what we talked about in class?

    6. You can find videos on platforms like YouTube, or on educational websites that offer courses on web development and CSS.
